# Quick Installation

## React

```bash
$ cd ./react/
```

### React Install

```bash
$ npm install
```

or

```bash
$ yarn install
```

or

```bash
$ pnpm install
```

### React Development

```bash
$ npm dev
```

or

```bash
$ yarn dev
```

or

```bash
$ pnpm dev
```

### React Build

```bash
$ npm build
```

or

```bash
$ yarn build
```

or

```bash
$ pnpm build
```
